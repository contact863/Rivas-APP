☁ Synchronisation en ligne (Firebase)
Non configuré — les données sont stockées localement.
Configurez Firebase pour partager les données entre tous les membres de l'équipe en temps réel.
Comment obtenir la configuration Firebase :
1. Allez sur console.firebase.google.com
2. Créez un projet → Realtime Database
3. Copiez les valeurs dans Paramètres du projet → Vos applications
